# 概要
- USB PD 12V DCjack
- 12v-5v, 12v-3.3v, 5v-1.2v DCDCコンバータ搭載
- USB3.0*4
- M.2 M key (SATA SSDのみ) 
- M.2 E key
- CANFD*2
- gigabit ethernet
- hd*iっぽいなにかの端子
- SDカードスロット
- IMU (BNO086)搭載
- stm32H723VGT6 搭載
- stm32 debugg用コネクタ
